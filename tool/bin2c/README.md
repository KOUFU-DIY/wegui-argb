# bin2c

`bin2c` 用来把 `input_bin` 目录中的多个 `.bin` 文件按文件名顺序合并，并生成：

- 一个合并后的二进制文件 `.bin`
- 一个 C 源文件 `.c`
- 一个头文件 `.h`

## 当前目录结构

- `bin2c.exe`
  Windows 下直接运行的主程序。
- `input_bin/`
  放原始 `.bin` 资源文件。
- `output/`
  默认输出目录。
- `builder/`
  构建 `bin2c.exe` 的工程目录。

## 默认行为

直接双击或运行：

```cmd
bin2c.exe
```

程序会：

1. 自动创建 `input_bin` 和 `output` 目录（如果不存在）
2. 扫描 `input_bin` 中所有 `.bin` 文件
3. 按文件名排序后依次合并
4. 在 `output` 中生成：

```text
merged_bin.bin
merged_bin.c
merged_bin.h
```

## 命令行参数

```text
--input DIR
    输入 bin 文件目录，默认: input_bin

--output-path DIR
--output-dir DIR
    输出目录，默认: output

--output-name NAME
--output-base NAME
    输出文件名主体，默认: merged_bin

--array-name NAME
    生成到 .c 中的数组名，默认: g_merged_bin_data

--symbol-prefix NAME
    生成到 .h 中的类型名前缀，默认: bin
```

## 使用示例

默认输出：

```cmd
bin2c.exe
```

自定义输出文件名：

```cmd
bin2c.exe --output-name res_pack
```

自定义输出目录：

```cmd
bin2c.exe --output-path D:\temp\out
```

同时自定义输出目录和文件名：

```cmd
bin2c.exe --output-name res_pack --output-path D:\temp\out
```

自定义数组名：

```cmd
bin2c.exe --array-name g_res_data
```

## 生成文件说明

### 1. `.bin`

输出一个按顺序拼接后的二进制文件，例如：

```text
merged_bin.bin
```

### 2. `.c`

生成一个 `unsigned char` 数组，例如：

```c
const unsigned char g_merged_bin_data[...] =
{
    ...
};
```

### 3. `.h`

头文件开头会带自动生成说明：

```c
/* This file was generated by bin2c. */
```

并生成以下内容：

- 资源 ID 枚举
- 资源大小枚举
- 资源地址宏
- 地址表 `unsigned long` 数组

示例：

```c
typedef enum
{
    IMG_A_ID = 0,
    IMG_B_ID = 1,
    bin_COUNT = 2
} bin_id_t;

typedef enum
{
    IMG_A_SIZE = 100,
    IMG_B_SIZE = 200,
} bin_size_t;

static const unsigned long bin_addr_table[bin_COUNT] =
{
    0,   //IMG_A
    100, //IMG_B
};

#define IMG_A_ADDR ((unsigned long)0)
#define IMG_B_ADDR ((unsigned long)100)
```

`bin_addr_table` 的顺序和 `XXX_ID` 的顺序一一对应。

## Windows 重新编译

如果修改了源码，可以重新编译：

```cmd
builder\build_win.cmd
```

编译成功后会在顶层生成新的：

```text
bin2c.exe
```

## macOS 构建

macOS 构建脚本在：

```text
builder/build_mac.sh
```

如果本机装好了编译环境，可执行：

```sh
sh builder/build_mac.sh
```

## 说明

- 输入文件会按文件名排序后再合并
- 输出地址顺序和 ID 顺序保持一致
- 如果 `input_bin` 中没有 `.bin` 文件，程序会提示后退出
